hexo 搭建随笔

hexo博客搭建

环境准备

  • node.js安装（自行官网）

  • git安装（自行官网）

检测node和git是否安装成功

  • sudo cnpm install –save hexo-deployer-git Deployment
    deploy：
    type: git
    repo: https://github.com/你的github名字/你的GitHub名字.github.io.git
    branch: master
    ```
  • 注意type;repo;branch所有冒号后加上空格

部署至GitHub

  • hexo d 中间需要输入一次GitHub的用户名和密码

查看部署效果

  • 访问地址：
    • https://你的GitHub名字.github.io

更换主题

以yilia主题为例

  • 下载主题（在blog根目录下操作）
  • 修改配置文件
    • 打开_config.yml，改成yilia
    • 注意冒号后加空格，养成良好的代码习惯，从我做起、
  • 效果查看
    • hexo c
    • hexo g
    • hexo s
